In a world where data privacy should be everyone's number one goal it can be difficult to share data without having to worry about accidentally exposing sensitive data. This is where Prospero come to help.
With prospero you can apply various masks to your data and share it knowing that your sensitive data is obfuscated how you want it.

Features

description

Read large CSV files and output them with masked data.

save

Save your masking settings to make it easy to re-mask the data in the future.

code

Use your masking settings and our CLI to script out masking jobs
